## Set GitHub Social Preview

After pushing the repository:

1. Open `Settings` → `Social preview`
2. Upload `docs/github-social-preview.png`
3. Save

The image depicts an LLM tensor/transformer stack being compressed into a
compact lossless archive, matching the theme of LLM lossless compression.
