| `github-wide-logo.png` | 1600×520 | README top banner |
| `github-social-preview.png` | 1280×640 | GitHub repository social preview |

## Set GitHub Social Preview

After pushing the repository:

1. Open `Settings` → `Options` → `Social preview`
2. Upload `docs/github-social-preview.png`
3. Save

The logo depicts an LLM data stream (left, circuit-style fan-in) being
losslessly compressed into a compact archive box (right) — matching the
benchmark's theme of LLM lossless compression.
